Introduction¶

This project uses the TMDb Movie Dataset that contains more than 10,000 records of movies with details on title, budget, revenue, cast, directors, popularities, etc. The dataset allows to explore a list of questions relating to:

  • The general trends are associated with movies over time?
  • The last question aims to explore the common traits among the profitable films regarding genre, actors, directors, production firms, budget and runtime.

Which genres are most popular from year to year? What kinds of properties are associated with movies that have high revenues?

def calculate_count(column):
    data = df_profit[column].str.cat(sep = '|')  # Convert column to string and seperate it by '|'
    data = pd.Series(data.split('|'))      # Storing the values seperately in a Pandas series
    count = data.value_counts(ascending = False)
    return count

Conclusions and limitations¶

Analysis allows to grasp following insights relating to a successful movies (profit >50 million dollars):

  • They are likely associated with genres: 'Action', 'Comedy', 'Adventure', 'Drama', 'Thriller'.
  • Casts are linked to: 'Tom Cruise', 'Tom Hanks', 'Brad Pitt', 'Adam Sandler', 'Cameron Diaz'.
  • Films are directed by directors: 'Steven Spielberg', 'Robert Zemeckis', 'Michael Bay', 'Quentin Tarantino', 'Roland Emmerich'
  • Production firms are associated with: 'Warner Bros.', 'Universal Pictures', 'Paramount Pictures', 'Twentieth Century Fox Film Corporation', 'Walt Disney Pictures'.
  • Average budget: Average budget of around 75 million dollars.
  • Run-time: the runtime is close to 2 hours (1.93 to be exact).
  • Films generates an average revenue of ~ 350 millions dollars.

Limitations

  • Only consider movies making a profit of 100 million dollars or over.
  • Rows with NaN were droped so some key data might be lost
  • These factors should only be taken into consideration but it does not 100% guarantee that success of a movie.
